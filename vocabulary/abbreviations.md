# Abbreviation

## Articles

| Abbr.  | Latin          | Meaning            |
| :----- | :------------- | :----------------- |
| e.g.   | exampli gratia | 例如               |
| etc.   | et cetera      | 等等（省略）        |
| et al. | et alia        | 等人，常用于文献作者 |
| i.e.   | id est         | 换句话说            |
| viz.   | videlicet      | 即，用于列举全部项目 |
| ibid.  | ibidem         | 同上               |
| V.V.   | vice versa     | 反之亦然           |
| iff.   | if and only if | 当且仅当           |
